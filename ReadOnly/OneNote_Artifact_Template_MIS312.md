# MIS312 OneNote Artifact Template (Standard Submission Format)

**Use this template for every MIS312 assignment and every Project02 milestone.**  
Create **one OneNote page per assignment**, then export it to **PDF** for submission.

---

## 1. Assignment Identification
- Course: **MIS312**
- Assignment / Milestone Name:
- Chapter (if applicable):
- Student Name:
- StarID:
- Date:
- Tools used (check all that apply): JupyterLab / VS Code / Raspberry Pi / Copilot / MIS312 AI Assistant

---

## 2. Task Summary (Student-written, 3–5 sentences)
In your own words:
- What did the assignment ask you to do?
- What was your plan?
- What result did you achieve?

---

## 3. Code Evidence (Paste only what matters)
Paste the most important code (not everything).

**Rules:**
- Use **monospaced** font
- Add short labels above each snippet (example: “Function definition”, “Loop”, “Sensor read”)
- If you used multiple files, include the key parts of each file

---

## 4. Execution Evidence (Screenshots REQUIRED)
Insert screenshots showing:
- The code running successfully
- The output/result in JupyterLab or VS Code terminal
- If you had an error, include a screenshot of the error **and** the fix

**Minimum expectation:** at least **1 screenshot** per assignment.  
**Project02 milestones:** at least **2 screenshots** (sensor output + your code).

---

## 5. AI Interaction (Exactly ONE meaningful interaction required)
Document one AI interaction that helped you learn.

Include:
- **AI Tool used:** (Copilot / MIS312 AI Assistant / ChatGPT)
- **Your prompt (copy/paste it):**
- **What the AI suggested (1–3 sentences):**
- **What you changed as a result (1–3 sentences):**

---

## 6. Reflection (Most important section)
Answer both:
1. **What did you change or correct after using AI or testing your code?**
2. **What do you understand now that you did not understand before?**

Write at least **4–6 sentences** total.

---

## 7. Export and Submit (Student checklist)
1. In OneNote, export/print this page as **PDF**.
2. Upload the PDF to the **D2L Assignment folder**.
3. Upload the same PDF to the **MIS312 AI Assistant** for feedback (if required).

**File size tip:** keep PDFs under **15 MB** by limiting large images and using standard screenshot size.
